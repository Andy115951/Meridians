# 中医经络腧穴可视化系统 产品文档

## 项目概述

本项目是一个基于 React + Vite 构建的中医经络腧穴可视化学习平台，旨在为中医学习者、从业者以及中医爱好者提供直观、交互式的经络穴位学习工具。

### 技术栈
- **前端框架**: React 19.2.3
- **构建工具**: Vite 6.2.3
- **样式方案**: TailwindCSS 3.4.0
- **图形渲染**: SVG

---

## 系统功能

### 核心功能
1. **人体经络可视化展示** - 支持正面/背面双视图
2. **交互式穴位探索** - 鼠标悬停显示穴位名称，点击查看详细信息
3. **三级缩放控制** - 全局视图/区域视图/详细视图
4. **经络路径渲染** - 不同颜色区分各条经络
5. **穴位信息面板** - 完整的穴位位置、功能、主治信息

### 交互特性
- 视图切换（正面/背面）
- 缩放级别控制
- 经络选择高亮
- 穴位悬停提示
- 点击穴位查看详情

## 身体区域划分

系统将人体划分为以下区域以便于穴位定位：

| 区域 | 范围描述 |
|------|----------|
| 头部 | 颈部以上区域 |
| 颈部 | 头部与胸部之间的连接区域 |
| 胸部 | 胸骨及肋骨区域 |
| 腹部 | 肚脐及周围区域 |
| 上肢 | 手臂区域 |
| 下肢 | 腿部区域 |

## 数据结构说明

### 经络数据结构
```javascript
{
  id: '经络唯一标识',
  name: '经络名称',
  chineseName: '经络中文名称',
  color: '显示颜色',
  description: '经络功能描述',
  path: {
    front: [{x, y}, ...],  // 正面路径坐标
    back: [{x, y}, ...]    // 背面路径坐标
  },
  points: [...]  // 穴位数组
}
```

### 穴位数据结构
```javascript
{
  id: '穴位代码 (如 LU1)',
  name: '穴位中文名称',
  pinyin: '拼音',
  englishName: '英文名称',
  x: 横坐标,
  y: 纺坐标,
  view: 'front/back',  // 所属视图
  major: true/false,   // 是否主要穴位
  location: '位置描述',
  functions: ['功能1', '功能2'],
  indications: ['适应症1', '适应症2'],
  note: '备注信息'  // 如原穴、募穴等
}
```
